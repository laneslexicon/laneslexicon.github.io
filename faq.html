Edward William Lane's Arabic-English Lexicon - the digital version

Q and A

  • Will it be free?
    Yes. It will be doubly-free: the sources will be released (free is in freedom) and it won't cost anything (free as in beer).
  • What will it run on?
    I'll be releasing installers for Windows and OSX together with build instructions for other platforms.
  • When will it be ready?
    In a couple of months, I hope. (Mind, I also said that a couple of months ago.)
  • What do I need to build a version for my OS?
    You'll need a copy of the Qt framework, an XSLT library and Perl (to build the database). The software is being developed under Debian Linux using free, opensource licensed tools.

Features

  • You can search for a root, a head word, or do a full-text search for an Arabic word (with or without diacritics) or an English word. If you know regular expression syntax, you can even search that way.
  • The text from the Lexicon can be copied and pasted into whatever you like.
  • Entries can be printed.
  • It can be customized - for example,if you don't like the Arabic font it ships with, you can use any font you want.
  • It comes with a virtual Arabic keyboard to help you enter Arabic text. If you know the Buckwalter transliteration, you can also enter Arabic that way.
  • It has simple note taking facility that allows you to attach your own comments to Lexicon entries.
  • Bookmarks, history, the usual stuff..
